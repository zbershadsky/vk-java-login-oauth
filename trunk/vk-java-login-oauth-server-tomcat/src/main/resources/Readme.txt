====== VK Java Login Running procedure on Tomcat application server ======

1. Run Tomcat
   * On the Windows platform
     Open a DOS prompt command and type the command
     vk.bat run
   * On Unix/linux/cygwin/MacOSX
     Open a terminal and type the command:
     ./vk.sh run
     You may need to change the permission of all *.sh files in the tomcat/bin dir by using: chmod +x *.sh 
2. Available services URLs:
   * Home page at the URL "http://localhost:8080/vk"
3. Default service credential: username is "vk" and password is "vk".
